## Proje Açıklaması

Bu proje, görüntü işleme ve derin öğrenme konularını öğrenmek amacıyla
geliştirilmiş bir **öğrenim (eğitsel) çalışmasıdır**.

MNIST veri seti kullanılarak eğitilen Evrişimsel Sinir Ağı (CNN) modeli ile,
webcam üzerinden kağıda yazılan rakamların gerçek zamanlı olarak tanınması
hedeflenmiştir.

Proje kapsamında CNN mimarisi, veri artırma (data augmentation),
OpenCV ile kamera kullanımı ve model tahmini süreçleri
uygulamalı olarak deneyimlenmiştir.
